# GisunOS Feature Breakdown

GisunOS provides a suite of professional workstation features ported to the web environment.

## 🖥️ Desktop Interface
*   **Dynamic Backgrounds**: Support for high-resolution static images and animated glass backgrounds.
*   **Application Dock**: MacOS-style dock with bounce animations and active indicator lights.
*   **Status Bar**: Global system indicators for time, battery, network, and active notifications.
*   **Workspace Management**: Support for multiple virtual desktops (coming soon).

## 🛡️ Security & Privacy
*   **Virtual Firewall**: Real-time monitoring of all app-originated network requests.
*   **Sandboxed Apps**: Higher-tier apps run in isolated iframes to prevent DOM-injection attacks.
*   **Privacy Dashboard**: View and revoke permissions for any installed application.

## 🎵 Media Hub (Music Hub)
*   **Ad-Free Experience**: Optimized for pure, uninterrupted listening.
*   **Visualizer**: High-performance audio spectrum visualizers.
*   **Playlist Management**: Integration with Supabase Storage for custom libraries.

## 🔍 SearchHub
*   **Global Search**: Integrated search for files, applications, and system settings.
*   **Quick Actions**: Execute commands (like "empty trash" or "lock screen") directly from the search bar.
*   **External Integration**: Search the web directly from the workstation interface.

## 💻 Developer Tools
*   **Terminal**: A UNIX-like terminal terminal for system control.
*   **Internal Explorer**: View and modify system configuration variables in real-time.
*   **Logic Visualizer**: Real-time view of the event bus and state changes (Dev Mode only).

## 🛠️ App Launcher
*   **Folder Support**: Organize applications into logical groups.
*   **Dynamic Scaling**: Icons automatically adjust to your screen size and resolution.
*   **Third-Party Support**: Standardized API for loading custom web-based apps into the OS environment.
